# 🛒 E-Commerce Sales Dashboard (Excel)

## Project Overview
Interactive Excel dashboard analyzing 51,290+ orders across global regions, 
built using Pivot Tables, Charts, and Slicers.

## Key Insights
- Month-wise Sales & Profit trends
- Region-wise Sales breakdown (Africa, Central, Oceania, etc.)
- Customer Segmentation analysis
- Order Aging & Shipping analysis

## Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Charts, Combo Charts)
- Data: 51,290 rows × 21 columns

## Dashboard Features
- Dashboard shows month-wise sales & profit trends, region-wise sales comparison, order count by month, and product category breakdown. Includes interactive slicers for filtering by region and ship mode
- Dynamic filters using Slicers
